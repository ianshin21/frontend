자바스크립트 개요

자바스크립트

웹 브라우저에서 많이 사용되는 인터프리터 방식의 스크립트 언어(객체지향)
- 스크립트 언어 : 기본프로그램의 동작을 사용자의 요구에 맞게 수행되도록 해주는 용도
- 인터프리터 방식 : 컴파일 과정 없이 실행하면서 코드를 하나씩 읽어내는 방식 (코드에 문법 오류가 있으면 실행시점에 알 수 있다.)

1. 자바스크립트 사용법

  1. inline : 태그 내에 직접 간단한 코드를 작성해서 실행되게 하는 방법
  2. internal : 문서내에 script 태그 영역에 소스코드를 작성하여 실행되게 하는 방법
  3. external : 별도의 .js 파일로 작성해서 가져다가 실행되게 하는 방법

1) inline 방식

태그 내에서 직접 간단한 소스코드를 작성해서 실행되게 하는 방법
[표현법] < 태그 on이벤트명= "해당 이벤트 발생시 실행할 소스코드 >

2) internal 방식

script 태그 영역에 함수단위로 소스코드들을 작성해 놓고
이벤트 발생시 해당 함수 호출로써 실행하는 방식
- script 태그는 head 태그 body 태그 내에 다 작성이 가능하다.

3) external 방식

별도의 .js 파일로 작성해서 가져다 사용하는 방법

2. 데이터 입출력

1) 데이터 입력

window.confirm()
- 어떤 질문에 대해 "예 / 아니오"의 결과를 얻을 때 사용
- 확인 버튼과 취소 버튼이 나타나며 확인 버튼을 클릭시 true, 취소 버튼 클릭시 false를 리턴한다.

window.prompt ()
- 텍스트 필드, 확인, 취소 버튼이 있는 대화 상자
- 입력한 메세지 내용을 리턴 값으로 돌려받는다.

2) 데이터 출력

a. document.write()
- 브라우저 화면상의 페이지에 직접 값을 출력한다.

b. window.alert()
- 브라우저 화면상에 알림창을 출력한다.
- window 객체는 모두 적용되는 것으로 생략 가능

window.alert()은 브라우저가 열리면 HTML문서(dom요소)가 로드되기 전에 가장 먼저 실행된다. 그러므로
이를 HTML문서(dom요소)가 로드된 후에 실행되게 하려면 window.onload / (document).ready() 함수내에 배치한다.

c. innerHTML로 요소 내용 변경

자바스크립트에서 태그 엘리먼트 값을 읽거나 값을 변경할 때 innerHTML 속성을 사용한다.

d. console.log()
- 개발자 도구 콘솔 화면에 출력하고자 할 때, 디버깅 할 때 주로 사용한다.
