이미지 관련 태그

src 속성

src 속성에 경로를 설정하여 이미지 파일을 불러온다.

alt 속성

사진의 경로가 잘못되거나 이미지를 제대로 표현할 수 없을 경우 대체 텍스트 용도로 사용
또는 시각장애인들을 위한 화면낭독기 프로그램에서 이미지를 대체하는 텍스트를 작성

width와 height

사진의 높이와 너비를 지정할 수 있다.
고정된 단위(px 등) 의크기로도 지정 가능하며
가변 크기 단위(%) 크기로도 지정 가능

고정 크기 단위 ; 화면 사이즈가 줄어도 크기가 변하지 않는다.

가변크기 단위 : 화면 사이즈 혹은 기준 사이즈에 따라 크기가 변경된다.

이미지 구역을 2개로 나누어 각각 링트 설정
