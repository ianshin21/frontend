jQuery 개요

jQjuery의 주된 작업은 실행중에 문서의 특정 엘리먼트를 찾아 조작하는 것이다.

jQjuery 라이브러리 연결 방법

head 태그 내부에 script 태그를 사용해서 jQuery 관련 .js 파일을 가져온다.
<< script type="text/javascript" src="JQuery js 파일.js"></script>>

1. jQuery 라이브러리 다운로드를 통한 연결 (오프라인)
- js 파일을 직접 다운로드 받아서 저장한 후 저장된 파일 경로를 지정하는 방법.

2. CDN(Contents Delivery Network)를 통한 연결 (온라인)
- 라이브러리를 제공하는 사이트의 주소를 이용하여 파일 경로를 지정하는 방법

* 3가지 버전 : jQjuery, 구글, MS 각 서버에 연결

jQuery 시작하기

1. jQuery는 문서에서 dom요소가 다 로드된 후에 실행되어야 한다.
- 자바스크립트에서의 해당 내용은 window.onload = function ( ) { 실행내용 } ;
해당 문서에서 단 한번만 작성 가능하다.
- jQuery에서는 $(document).ready(function ( ) { 실행내용 });
해당 문서에서 여러번 작성 가능하다. (그러나 한번만 사용하는 것을 지향한다.)

<교재 p499> HTML 문서를 로드 완료했을 때 호출된다.
아래 '문서의 DOM이 로드된 후 실행할 내용'이란 일테면
이미지를 불러오는 p태그가 있고 script 태그에 alert가 있을 때 창이 열리자 마자 alert이 실행되어
이미지가 로드 되지 않을 경우 이는 곧 dom 요소가 다 로드되지 않은 것이다.
이때 window.onload = function ( ) { alert() } ; 함수를 사용하여 dom이 완전히 로드된 후 이 함수내용이 실행되게 한다.

생각만큼 작동되지 않고 있음

2. jQuery에서 onload를 의미하는 구문 (3가지 방법)
- jQuery(document).ready(function(){ 문서의 DOM이 로드된 후 실행할 내용});
- $(document).ready(function(){ 문서의 DOM이 로드된 후 실행할 내용});
- $(function(){ 문서의 DOM이 로드된 후 실행할 내용});
3. 기본 작성 형태
$('선택자').메소드('속성명', '속성값'); $("p").css("color", "red");
